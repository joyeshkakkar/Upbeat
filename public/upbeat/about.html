What is Upbeat?

Upbeat is an endeavor at making an interactive web application utilizing information fetched from web API. The application was made as the final project for CS 5610 Web Development course under the guidance of Professor Jose Annunziato. This page uses API from Spotify to search for content and display it on the page. I have used MEAN(MongoDB, ExpressJS, AngularJS and Node.js), HTML5, CSS3, JavaScript, jQuery and Bootstrap to make and style this web application.



Spotify is a commercial music streaming service that provides digital rights management-restricted content from record labels including Sony, EMI, Warner Music Group, and Universal. Music can be browsed or searched by artist, album, genre, playlist, or record label.
